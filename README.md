# 1001-Albums-Journey

A personal dashboard to track your progress through the "1001 Albums You Must Hear Before You Die" challenge.

## Overview

This project provides a clean, modern interface to visualize your listening history, track your progress, and manage your daily album assignments. It connects with the [1001 Albums Generator](https://1001albumsgenerator.com/) to fetch your data.

## Features

-   **Dashboard**: View your current album assignment, progress overview, and quick stats.
-   **History**: Browse your complete listening history with cover art, ratings, and reviews.
-   **Stats**: Visual analytics of your ratings, genre distribution (planned), and listening habits.
-   **Music Integration**: Toggle between Spotify and Apple Music links.
-   **Responsive Design**: A sleek, dark-themed UI that works on desktop and mobile.

## Tech Stack

-   **Frontend**: React (Vite)
-   **Styling**: Vanilla CSS with a focus on glassmorphism and modern aesthetics.
-   **Routing**: React Router
-   **Icons**: Lucide React

## Getting Started

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```

2.  Navigate to the project directory:
    ```bash
    cd 1001-albums-journey/1001-albums-ui
    ```

3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or similar).

## Usage

1.  Open the app in your browser.
2.  Enter your **Project ID** from 1001albumsgenerator.com (found in your unique URL).
3.  Click "Connect Project" to load your data.
